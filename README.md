# json-to-interface
Generate TypeScript interface from JSON

## How to run

    ./node_modules/.bin/tsc --lib es6 print-inferred-types.ts
    ./node_modules/.bin/tsc --lib es6 print-types.ts
    node print-inferred-types.js sample.ts
